# Editorial calendar

## Quarterly review cycle

Every studio in the directory is re-verified once per quarter. The `lastVerified` front matter field tracks the date. If a studio doesn't respond to verification outreach in 2 attempts over 2 weeks, a banner is added to the page: *"No hemos podido verificar los datos de este estudio en la última actualización. La información puede estar desactualizada."*

| Quarter | Target review date | Studios to re-verify |
|---|---|---|
| Q2 2026 | 2026-06-30 | All (first cycle) |
| Q3 2026 | 2026-09-30 | All |
| Q4 2026 | 2026-12-31 | All |

## Sponsor outreach trigger

Start sponsor outreach when **both** conditions are met:
1. The site appears in top 5 of Google.es for ≥2 of: `estudio de podcast Valencia`, `alquiler estudio podcast Valencia`, `podcast studio Valencia`
2. Google Search Console shows ≥200 clicks/month

Target date: ~day 60–90 from first live deploy (estimated Q3 2026).
